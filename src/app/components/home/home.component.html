Super Héroes

En este sitio web encontrarás información sobre los super héroes de Marvel y DC muy famosos, podrás ver cada una de sus fichas con información detallada de cada personaje.

Esta aplicación fué desarrolada con la tecnología de Angular, con la idea de fabricar un sitio web SPA o SinglePageAplication de esta forma no necesitas acualizar o refrescar el navegador para mostrar contenido nuevo.
